# Presentation PDF assets

Put your **property logo** here (or pass any path on disk to `run-presentation-batch.sh`).

Supported formats for placement in Illustrator: **PNG**, **JPEG**, **PDF**, **AI**, **EPS** (avoid SVG unless you have verified Illustrator placement on your Mac).

The logo is scaled to fit within roughly **200 × 44 pt** in the header (upper-right) when using the **built-in layout**. With a **template `.ai`**, logo size follows your **`FP_LOGO`** frame — see **`templates/README.md`**.
